2d игра в изометрии. Игрок управляет девочкой. Необходимо дойти из одного угла комнаты в другую.
На стенах комнаты расположены пушки, которые стреляют огненными шарами. Так же есть патруль, который
ходит по заданному маршруту. В дополнении ко всему - летает шар убийца.
Левая кнопкая мышки - передвигает человечка.
Правая кнопка мышки - ставит/удаляет препятствие.
